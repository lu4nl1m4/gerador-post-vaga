# Gerador de Post de Vaga

Aplicacao web local para gerar artes de vaga em dois formatos: story 1080x1920 e feed 1080x1350, com campos editaveis de vaga, local, requisitos, tamanho da fonte dos requisitos, titulo do e-mail, data limite opcional e e-mail.

## Como abrir

```powershell
node server.js
```

Depois acesse:

```text
http://127.0.0.1:4173
```

Os botoes **Baixar Story** e **Baixar Feed** exportam as artes finais nos tamanhos corretos.
